## Data Flow

1. **User attempts dashboard navigation:**
   - Middleware catches request, refreshes session if valid, redirects to login if expired or invalid.
2. **Dashboard Shell mounts:**
   - Client fetches `/api/auth/me` via global Identity Provider context.
   - User identity, tenant details, and role list are cached in client state.
3. **Sidebar navigation is filtered:**
   - Sidebar filters `NAV_GROUPS` items to only display pages matching user's specific role (`owner`, `tenant_admin`, `teacher`, `student`, `parent`).
4. **Data queries run under row-level security (RLS):**
   - Direct db access or API handlers execute database calls with context-tied user tokens, fully enforcing multi-tenant isolation.
